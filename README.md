# File Manager

## About:

Basic file manager written in Blazor Server.

Works on Windows, Linux and MacOS.

Designed for remote admin of my personal server. 

Has no auth at all.

Wouldn't advise using in production. Could be unsafe with operations.

## Features:
- Table, List and Grid view
- Rename
- Delete
- Cut and Copy files
- Windows Explorer like selection with ctrl / shift etc
